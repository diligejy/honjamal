# 혼자말 (honjamal)

> **"힘든 날, 책이 건네는 한 줄"**  
> 일상의 지친 마음을 책 속의 문장으로 위로하는 미니멀 웹 서비스입니다.

## 🌿 핵심 가치
- **따뜻한 연결**: 복잡한 기능 대신, 문장 그 자체가 주는 힘에 집중합니다.
- **가독성 최우선**: 모든 디자인 수치는 인간 공학적 데이터와 접근성 표준을 준수합니다.
- **무비용/무백엔드**: 서버 없이도 어디서나 동작하며, 누구나 쉽게 접근할 수 있습니다.

## 🛠 기술 스택
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Data**: JSON (embedded for local compatibility)
- **Design Strategy**: TDD(Test-Driven Development) 기반 UI/UX 고도화

## 📈 개선 이력 (TDD Verified)
- **Accessibility**: 텍스트 대비비 4.5:1 확보 (WCAG AA 기준)
- **Typography**: 가독성 최적화 행간(1.6) 및 폰트 스케일링 적용
- **Interaction**: 0.8s Soft-Reveal(Blur) 애니메이션 구현
- **Stability**: 로컬 환경 CORS 이슈 해결을 위한 데이터 구조 최적화

## 📖 사용 방법
1. index.html 파일을 브라우저로 엽니다.
2. [문장 읽기] 버튼을 눌러 무작위로 건네지는 위로의 문장을 마주합니다.
